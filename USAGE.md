# 占用查看 使用说明

## 快速开始

### 1. 环境要求
- Android 5.0+ 设备
- 至少100MB可用存储空间
- 必要的权限授权

### 2. 安装步骤
1. 下载最新的APK文件
2. 在设备上允许"未知来源"安装
3. 安装应用
4. 授予必要权限

### 3. 基本使用
实时查看系统资源占用情况，帮助优化设备性能

### 4. 常见问题

**Q: 应用闪退怎么办？**
A: 请检查设备兼容性和权限设置

**Q: 功能无法正常使用？**
A: 请确保授予了所有必要权限

### 5. 技术支持
如有问题，请提交Issue到项目仓库
